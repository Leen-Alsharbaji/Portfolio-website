AI Drone

This simulation showcases a military surveillance drone operating in a virtual 3D environment, autonomously patrolling and capturing real-time footage using Unity. The drone streams images to a Python server, where YOLOv8 performs real-time person detection and sends bounding box data back to Unity for visualization. The system mimics real-world surveillance workflows, combining AI-driven navigation, obstacle avoidance, and object detection for defense and security applications.

Terrain Generation

This project features a procedural terrain generator powered by Perlin noise algorithms, which dynamically constructs realistic 3D landscapes with natural elevation variations. The system intelligently applies height-based texture mapping to simulate biome-specific features—such as grass, sand, rock, and water—enhancing visual fidelity through gradient-aware material blending. By integrating noise-based terrain synthesis with physics-compliant erosion and sedimentation models, the generator produces geomorphologically plausible environments for immersive military or surveillance training scenarios.
